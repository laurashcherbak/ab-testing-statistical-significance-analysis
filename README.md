# 📊 A/B Testing Statistical Significance Analysis

## 🎯 Project Overview
This project analyzes A/B testing experiment results using Python statistical methods and Tableau visualizations.  
The workflow automates significance calculations for multiple conversion metrics and visualizes experiment performance across test groups.

---

## 🧠 Project Goal
The goal of this project is to demonstrate:
- statistical analysis skills in Python
- understanding of A/B testing methodology
- automation of significance calculations using loops and arrays
- dashboard development in Tableau
- ability to interpret experiment performance and conversion metrics

---

## 🛠 Tools Used
- Python (Pandas, NumPy, SciPy)
- Google Colab
- Tableau Public

---

## 📌 Key Metrics
The project evaluates statistical significance for:
- add_payment_info / session
- add_shipping_info / session
- begin_checkout / session
- new_accounts / session

Additional analysis includes:
- country-level segmentation
- device analysis
- experiment-level comparison
- conversion behavior analysis

---

## 📊 Statistical Analysis
The Python workflow includes:
- automated metric processing using loops
- conversion rate calculations
- hypothesis testing
- p-value analysis
- significance flag generation
- preparation of final datasets for visualization

---

## 📈 Dashboard
The Tableau dashboard includes:
- conversion metric comparison
- significance indicators
- experiment filtering
- visualization of A/B test performance
